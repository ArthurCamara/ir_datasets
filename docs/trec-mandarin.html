ir_datasets: TREC Mandarin

"trec-mandarin"

A collection of news articles in Mandarin Chinese, used for multi-lingual evaluation in TREC 5 and TREC 6.

docs

Document fields: doc_id, text, marked_up_doc

Example

import ir_datasets
dataset = ir_datasets.load('trec-mandarin')
for doc in dataset.docs_iter():
    doc # namedtuple<doc_id, text, marked_up_doc>
