ir_datasets: ClueWeb09

"clueweb09"

ClueWeb 2009 web document collection. Contains over 1B web pages, in 10 languages.

The dataset is obtained for a fee from CMU, and is shipped as hard drives. More information is provided here.

  • Document collection site
docs
Document type:
WarcDoc: (namedtuple)
  1. doc_id: str
  2. url: str
  3. date: str
  4. content_type: str
  5. http_headers: str
  6. body: bytes

Example

import ir_datasets
dataset = ir_datasets.load('clueweb09')
for doc in dataset.docs_iter():
    doc # namedtuple<doc_id, url, date, content_type, http_headers, body>

"clueweb09/en/trec-web-2009"

The TREC Web Track 2009 ad-hoc ranking benchmark. Contains 50 queries with deep relevance judgments.

  • Shared task site
  • Shared task paper
queries
Query type:
TrecWebTrackQuery: (namedtuple)
  1. query_id: str
  2. query: str
  3. description: str
  4. type: str
  5. subtopics: Tuple[
    TrecSubtopic: (namedtuple)
    1. number: str
    2. text: str
    3. type: str
    , ...]

Example

import ir_datasets
dataset = ir_datasets.load('clueweb09/en/trec-web-2009')
for query in dataset.queries_iter():
    query # namedtuple<query_id, query, description, type, subtopics>
docs
Document type:
WarcDoc: (namedtuple)
  1. doc_id: str
  2. url: str
  3. date: str
  4. content_type: str
  5. http_headers: str
  6. body: bytes

Example

    doc # namedtuple<doc_id, url, date, content_type, http_headers, body>
qrels
Query relevance judgment type:
TrecQrel: (namedtuple)
  1. query_id: str
  2. doc_id: str
  3. relevance: int
  4. iteration: str

Example

    qrel # namedtuple<query_id, doc_id, relevance, iteration>
